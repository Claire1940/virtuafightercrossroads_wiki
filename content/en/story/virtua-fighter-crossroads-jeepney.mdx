<Section icon={Sparkles} title="Combat Mechanics and System Evolution">

While the core of Virtua Fighter remains the "Punch, Kick, Guard" system, Virtua Fighter Crossroads introduces several "Uprising" mechanics designed to make the 3D combat more cinematic and accessible without sacrificing depth.

<Steps>
  <Step num="1" title="Break & Rush System">
    Attacks targeting specific body regions (arms, legs, torso) accumulate "Break" damage. Once a threshold is reached, players can trigger a **Break Attack**, causing status effects that persist across rounds.
  </Step>
  <Step num="2" title="Flow Guard">
    A new defensive mechanic that emphasizes fluid motion. Unlike a static block, Flow Guard allows players to redirect the momentum of an incoming strike, opening up a window for a **Stunner** counter-attack.
  </Step>
  <Step num="3" title="Stun Combo Flow">
    Crossroads moves away from complex juggle-heavy inputs to a "Stun Combo" system. Landing a Stunner allows for intuitive follow-up sequences that feel more like a choreographed action movie.
  </Step>
</Steps>

| System | Category | Primary Function |
| :--- | :--- | :--- |
| **Standard Rules** | **Classic** | 3-round matches, health depletion, ring-outs |
| **Uprising Rules** | **New Mode** | Momentum-based scoring and dynamic shifts |
| **Cinematic Hits** | **Presentation** | Action-movie camera angles for decisive blows |
| **Intuitive Controls** | **Accessibility** | Simplified inputs for complex legacy maneuvers |

<Callout type="info" title="Technical Foundation">
The game still adheres to the "Strike beats Throw, Throw beats Guard, Guard beats Strike" triangle. New systems like **Flow Guard** add layers to this foundation rather than replacing it.
</Callout>

</Section>
