| 버튼 종류 | 기본 입력 | 전투 시 기능 | 플레이어 영향 |

| **펀치 (P)** | Square / X | 빠른 타격, 잽, 콤보 시동기 | 빠른 속도, 낮은 리스크 |
| **킥 (K)** | Triangle / Y | 긴 리치 타격 및 강력한 피해 | 보통 속도, 긴 사거리 |

| **P+G** | L1 / LB | 가드를 굳힌 상대를 징벌하는 기본 잡기 | 니가와(버티기) 대처 |

<Section icon={Target} title="컨트롤러 설정 및 최적화">

**버추어 파이터 크로스로드 버튼 레이아웃**의 최적화는 선호하는 하드웨어에 따라 다릅니다. 이 게임은 가정용 콘솔로 2027년 출시를 목표로 하고 있기 때문에, 기본 매핑은 PS5 듀얼센스(DualSense) 및 Xbox Series 컨트롤러에 맞춰 구축되었습니다. 하지만 전투 시스템의 깊이를 더하기 위해 커스텀 매크로 매핑을 활용하는 것이 크게 도움이 됩니다.

| 컨트롤러 구성 요소 | 권장 매핑 | 선택 이유 |
| :--- | :--- | :--- |
| **전면 버튼** | P, K, G, P+K | 빠른 접근을 위한 클래식 다이아몬드 레이아웃 |
| **우측 트리거** | P+K+G (매크로) | 범용적이고 강력한 기술에 쉽게 접근 가능 |
| **좌측 트리거** | P+G (잡기) | 잡기 풀기를 즉각적으로 수행할 수 있도록 보장 |
| **D-패드 / 아날로그 스틱** | 이동 | 거리 조절을 위한 정밀한 8방향 이동 |
| **터치패드 / 셀렉트 버튼** | 연습 초기화 | 새로운 배틀 모드 연구소(연습실) 필수 기능 |

아케이드 스틱을 사용하는 플레이어에게는 '타입 A' 레이아웃이 여전히 표준입니다. 이는 펀치, 킥, 가드를 삼각형이나 일렬로 배치하여 복잡한 **스턴 콤보** 중에 손가락을 굴려 가며 입력하는 '피아노 연주식' 입력을 가능하게 합니다.

<Callout type="success" title="전략 노트">
개발팀은 조작 철학을 '직관적'으로 재구축하고 있습니다. 이는 신규 유저들의 진입 장벽을 낮추기 위해 가정용 콘솔 버전에서 여러 버튼 동시 입력(예: P+K+G)이 기본적으로 단일 트리거에 매핑되는 경우가 많음을 의미합니다.
</Callout>

  <Step num="3" title="스터너 연계하기">

  </Step>
  <Step num="4" title="플로우 가드 반복 훈련">
