<Section icon={Swords} title="戦闘メカニクスとシステムの革新">

**バーチャファイター クロスロード ウィッシュリスト**の核心は、伝統的な「ジャンケン（打撃・投げ・ガード）」の基礎を失うことなく、いかにしてテクニカルな3D戦闘を現代化するかという点にあります。RGGスタジオは、過去の静的なフレームデータ重視の戦いを超え、試合の勢いを変化させるいくつかの新システムを導入しています。特に「ブレイク＆ラッシュ」システムは議論の的となっており、プレイヤーは特定の部位を狙ってダメージを蓄積させ、それがラウンドをまたいで影響を与える仕組みになっています。

<Callout type="warning" title="戦闘の変革">
「アップライジング・ルール」の導入は、従来のシリーズで見られた「スタンダード・ルール」とは大きく異なり、よりダイナミックで勢いに基づいたスコアリングシステムを示唆しています。
</Callout>

ファンは、過去の知識と現代的な創造性の両方が報われるシステムを望んでいます。「スタナー」と「スタンコンボ」のメカニクスは、初心者にはコンボルートを直感的に理解させ、ベテランには防御の堅い相手を崩す新しい手段を提供するように設計されています。

<Section icon={Shield} title="オンラインインフラと競技機能">

競技コミュニティにとって、**バーチャファイター クロスロード ウィッシュリスト**の最も重要な部分はオンライン体験です。2027年のリリースに向けて構築されている以上、ディレイ方式のネットコードは許容されません。プレイヤーは、『バーチャファイター』のシビアなフレーム入力がオンラインでも通用するよう、最先端のロールバック・ネットコードを求めています。

<Callout type="warning" title="競技スタンダード">
『ストリートファイター6』や『鉄拳8』といった現代の格闘ゲームは、オンラインロビーやランクシステムの基準を非常に高く設定しています。『クロスロード』が生き残るためには、これらに匹敵する必要があります。
</Callout>

主なオンライン機能のリクエスト：
- **クロスプラットフォームプレイ**: PC、PlayStation、Xboxのプレイヤーが一つの統合されたプールで競い合えること。
- **充実したトレーニングモード**: フレンドと一緒にコンボやフレームデータの練習ができるオンライン「ラボ」。
- **トーナメント統合**: ローカルおよびグローバルなトーナメントを運営・管理するためのゲーム内ツール。
- **観戦ツール**: ストリーマーやコメンテーターがシネマティックな演出を際立たせるための高度なカメラオプション。

| 機能 | 重要度 | コミュニティの総意 |
| :--- | :--- | :--- |
| **ロールバック・ネットコード** | 致命的 | 競技の公平性のために必須 |
| **クロスプレイ** | 高 | 長期的なプレイヤーベースの維持に不可欠 |
| **ランクリーグ** | 高 | 透明性のあるMMRと報酬が必要 |
| **リプレイシアター** | 中 | 研究やSNSでの共有に有用 |
