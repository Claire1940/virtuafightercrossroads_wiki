<Section icon={Shield} title="ゲームプレイメカニクスと戦闘システム">

『**バーチャファイター クロスロード**』の戦闘は、シリーズ伝統の「打撃・投げ・ガード」の三すくみを維持しつつ、現代的な3D格闘体験を提供するためにいくつかの新要素を導入しています。主なルールセットである「**スタンダード**」と「**アップライジング**」の2つにより、異なる感覚で戦闘を楽しむことができます。

| システム | カテゴリ | 仕組み |
| :--- | :--- | :--- |
| **ブレイク＆ラッシュ** | 攻撃的 | 特定の部位にダメージが蓄積し、「ブレイク」攻撃を誘発。 |
| **フロウガード** | 防御的 | ブレイク＆ラッシュをいなす、流動的な防御アクション。 |
| **スタナー** | 打撃 | 相手をスタンコンボの状態にする特殊な攻撃。 |
| **スタンコンボ** | コンボ | 映画のような大ダメージを与える、簡略化されつつも深いコンボ。 |
| **アップライジング・ルール** | バトルルール | 従来のラウンド制とは異なる、勢い（モメンタム）重視のルール。 |

「ブレイク＆ラッシュ」システムは特に革新的で、腕や足などの特定の部位へのダメージがマッチ内のラウンドをまたいで引き継がれます。これにより、プレイヤーは目先の体力だけでなく、長期的な戦略を考える必要に迫られます。

<CardGrid cols={3}>
  <Card title="三すくみ">
    **打撃**は投げに勝ち、**ガード**は打撃を防ぎ、**投げ**はガードを崩します。
  </Card>
  <Card title="シネマティクス">
    新しいカメラワークにより、フレームデータの精度を損なうことなく、決定打に映画のような迫力を与えます。
  </Card>
  <Card title="アクセシビリティ">
    操作系は初心者にとって直感的であると同時に、上級者の読み合いの深さを維持するように再構築されています。
  </Card>
</CardGrid>

<Callout type="info" title="メカニクスの更新">
過去作とは異なり、『**バーチャファイター クロスロード**』のストーリーモードには多人数との乱闘や巨大なボス戦が含まれています。これらの遭遇戦は1対1の戦闘と同じテクニカルなエンジンを使用していますが、異なる空間把握能力が求められます。
</Callout>

<Section icon={Sparkles} title="初心者ガイド：基本の習得">

**バーチャファイター クロスロード Switch 2**の発売に備えるなら、基本をマスターすることが不可欠です。バーチャファイターは飛び道具に頼らない、間合いとタイミングのゲームであるという点で独特です。

<Steps>
  <Step num="1" title="ボタンを覚える">
    3つの基本入力、**パンチ (P)**、**キック (K)**、**ガード (G)** に集中しましょう。ほとんどの複雑な技は、これら3つの組み合わせです。
  </Step>
  <Step num="2" title="三すくみの練習">
    チュートリアルモードで、ガードが固い相手には投げを、投げようとする相手には打撃を当てる練習をしましょう。
  </Step>
  <Step num="3" title="移動のマスター">
    3Dの動きは非常に重要です。サイドステップを使って直線的な打撃を避け、リングアウトを避けるために常にリングの中央を維持しましょう。
  </Step>
  <Step num="4" title="上・中・下段の理解">
    立ちガードは上段と中段を防ぎます。しゃがみガードは下段を防ぎます。覚えておいてください：中段攻撃はしゃがんでいる相手に当たるため、最も危険な攻撃の一つです。
  </Step>
</Steps>

<Callout type="tip" title="セーフプレイ">
バーチャファイターにおいて「セーフ（確反なし）」とは、攻撃がガードされても簡単に反撃を受けないことを意味します。初心者は、大きな反撃のリスクを負わずにプレッシャーを与え続けられる、隙の少ない中段攻撃に焦点を当てるべきです。

<Section icon={Trophy} title="バトルモードとオンライン接続">

ストーリー以外にも、『**バーチャファイター クロスロード**』には充実した対戦モードが用意されています。ローカルプレイでもランクマッチでも、長期的な競技コミュニティをサポートするように設計されています。

| モード | 形式 | 主な特徴 |
| :--- | :--- | :--- |
| **バトルモード** | 1v1 対戦 | オフラインおよびオンラインでの伝統的な対戦。 |
| **オンラインランク** | 競技 | グローバルリーダーボードを備えたスキルベースのマッチメイキング。 |
| **カジュアルロビー** | 交流 | 友達との練習や、リアルタイムでの試合観戦。 |
| **アーケードモード** | ソロ | CPUキャラクターを相手にするクラシックな勝ち抜き戦。 |
| **トレーニングモード** | 練習 | 詳細なフレームデータとコンボ記録ツール。 |

セガはこれらのモードにおいて「アクション映画」のような演出を優先しており、標準的な試合でも高い緊張感を感じられるようにしています。オンラインプレイへの**アップライジング・ルール**の導入により、従来の**スタンダード・ルール**に比べてよりダイナミックな対戦メタが生まれることが期待されています。
