<Section icon={Target} title="ストーリーモードと都市ヴィラサパラ">

バーチャファイター クロスロードは、シングルプレイヤーの**ストーリーモード**に重点を置くことで、シリーズの伝統を打ち破ります。物語は、武術が主要な文化的・経済的推進力となっている架空の東南アジアの都市**ヴィラサパラ**で展開されます。「アルマ・カルタ（Arma Carta）」の下で銃器が事実上禁止された結果、紛争は公認の試合や地下の乱闘を通じて解決される社会となっています。

都市はいくつかの特徴的な地区に分かれており、それぞれ独自のサイドクエスト、武術の道場、ストーリーイベントが用意されています。

| 地区名 | 雰囲気 | 主な特徴 |
| :--- | :--- | :--- |
| **シティセンター** | 城壁に囲まれた / 高い警備 | バト大統領の権力の拠点であり、ヴィラ・ファイト・フェストの中心地。 |
| **歓楽街** | ネオン / 混雑 | 地下の闘技場と街のナイトライフの本拠地。 |
| **リゾートゾーン** | 豪華 / 開放的 | 主要な大会期間中に国際的なファイターが滞在する場所。 |
| **マカリア** | 工業的 / 荒廃 | 犯罪組織の拠点であり、バクナワ・キラー調査の重要地点。 |

<Section icon={Sparkles} title="バトルメカニクスとゲームプレイの進化">

「パンチ、キック、ガード」という核となる哲学は維持しつつ、**バーチャファイター クロスロードのSteam**プレイヤーは、競技的な深みとシネマティックな演出を両立させるために再構築されたバトルシステムを体験することになります。龍が如くスタジオは、「格闘アドベンチャー」の構造を支えるために、いくつかの新しいレイヤーを戦闘に導入しました。

| システム | カテゴリー | メカニクスの詳細 |
| :--- | :--- | :--- |
| **ブレイク＆ラッシュ** | 攻撃 | 部位ダメージが蓄積。「ブレイク」攻撃の効果はラウンドを跨いで継続。 |
| **フローガード** | 防御 | スタイリッシュで流れるようなパリィシステム。正確なタイミングで視覚的な演出と共に有利を得る。 |
| **スタナー** | 打撃 | 新しい「スタンコンボ」の流れを開始する特殊な攻撃。 |
| **アップライジング・ルール** | バトルルール | 通常のラウンドルールとは異なる、勢いに基づくスコアリングシステム。 |
| **スタンコンボ** | コンボ | 複雑な入力の壁を取り除いた、簡略化された追撃システム。 |

<Callout type="tip" title="戦略のヒント">
**ブレイク＆ラッシュ**システムを使用して、複数のラウンドにわたり相手の特定の部位を狙いましょう。一度部位が「破壊（Broken）」されると、その部位の防御オプションや攻撃速度に大きなペナルティが課せられます。
</Callout>

また、ストーリーモードでは**多人数乱闘（Multi-Opponent Brawls）**も特徴で、集団戦を処理するために調整されたバーチャファイターエンジンが使用されます。一方で、伝統的な1対1の体験は**バトルモード**で維持されており、通常ルールとアップライジング・ルールの両方でローカルおよびオンライン対戦をサポートしています。

</Section>

<Section icon={Shield} title="初心者ガイド：基本のマスター">

シリーズの新規プレイヤーや、他の龍が如くスタジオ作品から移行するプレイヤーのために、バーチャファイター クロスロードは直感的なエントリーポイントを提供しています。遠距離からの飛び道具による牽制よりも、近距離での読み合いに焦点が当てられています。

<Steps>
  <Step num="1" title="三すくみを学ぶ">
    **打撃は投げに勝ち**、**ガードは打撃に勝ち**、**投げはガードに勝つ**ことを理解しましょう。この基本的なループがあらゆる攻防の核心です。
  </Step>
  <Step num="2" title="移動をマスターする">
    バーチャファイターは3Dゲームです。直線的な打撃を避けるためにサイドステップを練習し、環境を利用してリングアウトのチャンスを作りましょう。
  </Step>
  <Step num="3" title="スタナーを練習する">
    スタナーは大ダメージへの入り口です。**スタンコンボ**の窓を開くためのキャラクターごとの特定の入力を覚えましょう。
  </Step>
  <Step num="4" title="フローガードを活用する">
    単にガードボタンを押し続けるだけでなく、防御入力をタイミングよく行い、反撃のために大きなフレーム有利を得られるフローガードの発動を狙いましょう。
  </Step>
  <Step num="5" title="ストーリーから対戦へ">
    ストーリーモードを使って4人の主人公に慣れましょう。ヴィラサパラの街頭で学んだ教訓は、競技的なバトルモードに直接活かすことができます。
  </Step>
</Steps>
