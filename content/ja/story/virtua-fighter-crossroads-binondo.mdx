<Section icon={Sparkles} title="戦闘メカニクスとシステム">

その根幹において、**『バーチャファイター クロスロード ビノンド』**はこのジャンルを定義したテクニカルな深みを保持しています。伝統的なP（パンチ）、K（キック）、G（ガード）のレイアウトは健在ですが、試合をよりダイナミックで視覚的に表現力豊かなものにするためのいくつかの新システムによって強化されています。

### 新たな攻防のレイヤー

最も重要な追加要素は**「ブレイク＆ラッシュ」**システムです。特定の部位を繰り返し攻撃することで、プレイヤーは「ブレイク」状態を引き起こすことができ、これはラウンド間でも引き継がれます。これにより、プレイヤーは単なる即時の体力減少だけでなく、長期的な戦略を考える必要に迫られます。これを補完するのが、ガードから反撃へのより流動的な移行を可能にする防御メカニクス**「フローガード」**です。

| 戦闘システム | カテゴリ | 仕組み |
| :--- | :--- | :--- |
| **スタンダードルール** | バトルルール | 伝統的な体力の削り合いとリングアウトによる勝利 |
| **アップライジングルール** | バトルルール | 試合の勢いに基づくスコアリングとダイナミックな変化 |
| **ブレイク＆ラッシュ** | 攻撃 | 部位にダメージが蓄積し、ブレイクを誘発 |
| **フローガード** | 防御 | 反撃の隙を作る、スタイリッシュなパリィのようなガード |
| **スタナー** | テクニック | 新しい「スタンコンボ」フローを開始する打撃 |
| **スタンコンボ** | システム | スタナー後の直感的で高ダメージな追撃 |

<Callout type="info" title="システムの習得">
スタンコンボシステムは、初心者が20フレームの完璧な入力を覚えることなく満足感のある連携を繰り出せるように設計されていますが、ベテランはタイミングによってダメージを最適化することも可能です。
</Callout>

</Section>

<Section icon={Shield} title="初心者ガイド：基本のマスター">

シリーズ初心者や久しぶりに復帰するプレイヤーにとって、**『バーチャファイター クロスロード ビノンド』**は難しく感じるかもしれません。「アニメ系格闘ゲーム」や飛び道具のある2Dゲームとは異なり、『バーチャファイター』はわずかな距離、フレームデータ、そして心理的な読み合いのゲームです。

<Steps>
  <Step num="1" title="三すくみを理解する">
    『バーチャファイター』はジャンケンの論理で動いています。打撃は投げに勝ち、投げはガードに勝ち、ガードは打撃に勝ちます。相手が何を多用しているかを見極め、それに応じて対処しましょう。
  </Step>
  <Step num="2" title="移動と間合いのマスター">
    3D空間を活用しましょう。直線的な攻撃を避けるにはサイドステップが不可欠です。攻撃が届き、かつ投げに反応できる「絶妙な距離感」を維持する練習をしてください。
  </Step>
  <Step num="3" title="キャラクターの「主力技」を覚える">
    一度にすべての技リストを覚えようとしないでください。一つの速い牽制技（通常はP）、一つの浮かせ技、そして一つの信頼できる投げ技に集中しましょう。複雑なスタンコンボに進む前に、この3つをマスターしてください。
  </Step>
  <Step num="4" title="トレーニングモードの活用">
    龍が如くスタジオは広範なチュートリアルシステムを実装しています。これを使ってフレームデータを可視化し、バクナワ・キラーのユニークな攻撃パターンに対する特定の防御反応を練習しましょう。
  </Step>
</Steps>

| 練習目標 | 推奨キャラクター | 理由 |
| :--- | :--- | :--- |
| **基本打撃** | シエロ | ボクシング主体の速い技はタイミングが取りやすい |
| **投げの駆け引き** | ウルフ・ホークフィールド | 投げ入力の受付時間が長く、高ダメージ |
| **防御の読み** | パイ・チェン | タイミングを学ぶのに適した優れた返し技と捌き技 |
| **精度の向上** | 結城 晶 | 正確な入力と間合いの重要性を教えてくれる |

<Callout type="tip" title="プロの戦略">
ガードに成功した後は、通常、こちらがフレーム的に有利です。これがあなたの「ターン」です。ためらって無駄にせず、すぐに速い中段攻撃を繰り出して、しゃがんだり投げようとしたりする相手を捉えましょう。
</Callout>
