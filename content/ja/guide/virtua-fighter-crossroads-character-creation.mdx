<Section icon={Map} title="ステップごとの構築手順">

virtua fighter crossroads character creation に取り組む最もきれいな方法は、段階ごとに組み立てることです。各段階の শেষেには、実戦で再現できる要素が1つ残るようにしましょう。ある選択が安定性を高めないなら、まだ残すには早すぎます。

<Steps>
  <Step num="1" title="メインを1人決める">
    最も楽しく使えて、何セッションも練習する姿を想像できるキャラクターを選びましょう。メインを1人に絞ることで学習の焦点が定まり、浅い進歩を防げます。
  </Step>
  <Step num="2" title="基本ゲームプランを決める">
    近距離での圧力、カウンターヒット、より安全な間合い管理のどれで勝ちたいかを決めます。攻めのパターンを1つ、防御の答えを1つ書き出しましょう。
  </Step>
  <Step num="3" title="信頼できる選択肢を3つに絞る">
    開始時の行動1つ、反撃用の行動1つ、逃げや立て直しの行動1つを選びましょう。信頼できる行動が3つあれば、不確かな行動を10個抱えるよりずっと良いです。
  </Step>
  <Step num="4" title="トレーニングと実戦で試す">
    まずはトレーニングでプランを実行し、その後で実戦に持ち込みます。プレッシャー下で機能するものは残し、何度も失敗するものは切りましょう。
  </Step>
  <Step num="5" title="振り返って磨く">
    各セッション後に、悪い癖を1つ消し、良い癖を1つ強めます。細かな修正の積み重ねが、常に組み直すよりも良いファイターを作ります。
  </Step>
</Steps>

<Callout type="info" title="実践ルール">
より良くガードできる、より早く反撃できる、落ち着きを保てるようになる——そのどれにも役立たない選択肢は、基本操作より優先度が低いです。
</Callout>

| フェーズ | 焦点 | 成功の目安 |
|---|---|---|
| 序盤 | メイン1人、プラン1つ | 何度も切り替えなくなる |
| 中盤 | 反撃のタイミング | 守りをダメージに変えられる |
| 終盤 | 対戦相手ごとの調整 | 焦らず対応できる |
| 継続 | 癖の整理 | 繰り返すミスが減る |

</Section>

<Section icon={Sparkles} title="カスタマイズの優先順位と注意点">

最高のキャラクターセットアップは、最も複雑なものではありません。すぐにプレイを良くしてくれるものこそが最良です。まずは、判断力と実行力を高める道具から始めましょう。派手な選択肢は、基本の動きが安定してから後で追加すれば十分です。

| 優先度 | 残すべき？ | 理由 |
|---|---|---|
| メインキャラの個性 | はい | 他の選択すべての土台になる |
| シンプルな反撃ルート | はい | ミスを価値に変えられる |
| 基本の圧力連携 | はい | 再現性の高い攻めを作れる |
| 見た目の変更 | 後で | 見た目より安定性のほうが重要 |
| レアなニッチ要素 | 後で | 基本が固まる前は使いにくい |

| よくあるミス | 起きること | より良い対処 |
|---|---|---|
| 選択肢を増やしすぎる | 試合中に迷う | 小さな道具セットに絞る |
| 早い段階で上級テクを追う | 実行が崩れる | まずは基本ルートを覚える |
| 守りを無視する | すぐ主導権を失う | 明確なガード→反撃プランを追加する |
| 毎回セットアップを変える | 進歩がランダムに感じる | 1つのセットを十分に試す |
