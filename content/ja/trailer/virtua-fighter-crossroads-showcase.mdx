<Section icon={Swords} title="戦闘メカニクスとバトルシステム">

**バーチャファイター クロスロード ショーケース**では、戦闘エンジンの全面的な刷新が詳細に説明されました。パンチ、キック、ガードという中核となる哲学は維持されつつも、「ファイティングアドベンチャー」のゲームプレイを促進するための新しいシステムが追加されました。最も注目すべき追加要素は、ラウンドをまたいでダメージが蓄積される「ブレイク＆ラッシュ」システムです。

<Steps>
  <Step num="1" title="3すくみをマスターする">
    「打撃は投げに勝ち、ガードは打撃に勝ち、投げはガードに勝つ」という基本ロジックを理解しましょう。この根本的なバランスこそが、本作の心臓部です。
  </Step>
  <Step num="2" title="ブレイク＆ラッシュを活用する">
    特定の身体部位を狙って「ブレイク」ダメージを蓄積させます。部位が破壊されると、そのマッチの残りのラウンド中、弱体化した状態が維持され、新たなコンボルートが開かれます。
  </Step>
  <Step num="3" title="フローガードを習得する">
    映画のような演出で攻撃をパリィする、新しい防御メカニクスを使用します。これにより戦況の主導権を引き戻し、反撃のための「スタナー」状態を作り出します。
  </Step>
  <Step num="4" title="スタンコンボを叩き込む">
    スタナー状態の相手に対して、直感的な新しいコンボシステムで追撃します。これにより、過去作のようなフレーム単位の正確な入力を必要とせず、高ダメージの連携が可能になります。
  </Step>
</Steps>

<Callout type="success" title="プロの戦闘のコツ">
「アプライジング・ルール」モードは、よりダイナミックな試合展開を求めるプレイヤー向けに設計されています。「スタンダード・ルール」とは異なり、アプライジングでは、アグレッシブで多様なプレイスタイルに報いる、戦況を左右するスコアリングが特徴です。
</Callout>

| メカニクス | タイプ | 効果 |
| :--- | :--- | :--- |
| **スタンダード・ルール** | バトルルール | 従来の体力減少とリングアウトによる勝利 |
| **アプライジング・ルール** | バトルルール | 劇的な試合展開をもたらすモメンタムベースのスコアリング |
| **フローガード** | 防御 | 映画のようなヒットリアクションを伴うアクティブパリィシステム |
| **スタナー** | 攻撃 | スタンコンボのチャンスを作り出す打撃技術 |
| **ブレイク＆ラッシュ** | 戦略 | ラウンドをまたいで蓄積される持続的な肢体ダメージ |

</Section>

<Section icon={Trophy} title="モードとオンラインプレイの解説">

**バーチャファイター クロスロード ショーケース**の最後のセグメントでは、プレイヤーがどのように時間を過ごすかに焦点が当てられました。ゲームは、大規模なシングルプレイヤー用の「ファイティングアドベンチャー」と、競技志向の熱狂的ファン向けの専用の「バトルモード」に分かれています。これらのモード間の移行はシームレスに設計されており、ストーリーモードでのキャラクターの成長が、オンラインプレイにおけるコスメティックアイテムのアンロックに引き継がれます。

オンラインバトルモードでは、スタンダード・ルールとアプライジング・ルールの両方が用意されており、セガは世界的な競争をサポートするための堅牢なネットコードを約束しています。「ファイティングアドベンチャー」側には、1対1の決闘だけでなく、複数の敵との乱闘や、ヴィラサパラの環境を利用した映画のようなボス戦も含まれています。

<Callout type="info" title="オンライン機能">
バトルモードには、ローカル対戦、オンラインランクマッチ、カジュアルロビーが含まれます。RGGスタジオは、発売後の健全な配信コミュニティを維持するために「クリエイター・ルール」も実装しています。
</Callout>

| モード | 形式 | 主な特徴 |
| :--- | :--- | :--- |
| **ストーリーモード** | シングルプレイヤー | 分岐するルート、4人の主人公、探索 |
| **バトルモード** | 1v1対戦 | オンライン/オフラインのランクプレイ、スタンダード/アプライジング・ルール |
| **ヴィラ・ファイト・フェスト** | トーナメント | 限定報酬が手に入るシーズンオンラインイベント |
| **トレーニングモード** | 練習 | フレームデータツール、コンボトライアル、チュートリアルルート |
| **ストリートファイト** | アドベンチャー | ヴィラサパラの各地区で発生する複数敵とのエンカウント |

</Section>
